🏏 Cricket Match Simulator (Python)
📌 About the Project

This is a simple terminal-based cricket match simulator built using Python.
It allows two teams to play a virtual match with toss, overs, runs, wickets, and live score updates.

🚀 Features
Toss system (Head/Tail)
Batting and bowling selection
Custom number of overs
Ball-by-ball input system
Runs, wickets, wides, no-balls handling
Automatic strike rotation
Player batting rotation system
Live scoreboard after every over
Simple ASCII-style scoreboard display
🧠 How It Works
User enters team names
Toss decides batting first
Match runs over user-defined overs
Each ball takes input:
0–6 → runs
W → wicket
WD → wide ball (+1 run)
NB → no ball (+1 run)
Batsmen rotate based on runs
Match ends when overs finish or 10 wickets fall
▶️ How to Run
1. Clone the repository
